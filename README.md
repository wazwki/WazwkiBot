# WazwkiBot

WazwkiBot is a telegram bot that using stack:
- Puython 3.12
- AioGram 3

## Installation

Use the git to install WazwkiBot.

```bash
git clone https://github.com/wazwki/WazwkiBot.git
```

## Usage

```python
python bot_app/bot.py
```
